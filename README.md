# URL Shortener Monorepo

This repository contains both backend and frontend projects:

- `backend/` - Spring Boot backend (Maven)
- `frontend/` - React + Vite frontend

## Backend

From the repository root:

```powershell
cd backend
.\mvnw.cmd spring-boot:run
```

Run tests:

```powershell
cd backend
.\mvnw.cmd test
```

## Frontend

From the repository root:

```powershell
cd frontend
npm install
npm run dev
```
